# Curso de Algoritmos — Gustavo Guanabara

> 🚧 **Projeto em andamento.** Estou no início do curso, então o repositório ainda está sendo construído aula a aula.

Repositório criado para documentar meus estudos no curso de **Algoritmos e Lógica de Programação**, do professor Gustavo Guanabara ([Curso em Vídeo](https://www.cursoemvideo.com/course/curso-de-algoritmos/)).

O curso trabalha lógica de programação usando **pseudocódigo**, praticado aqui através do app mobile Pseudocode, com o dialeto **VisuAlg:Português**, cobrindo desde variáveis e estruturas de decisão até vetores, matrizes e sub-rotinas.

## 📁 Estrutura do repositório

```
├── algoritmos/               # Algoritmos desenvolvidos durante as aulas
├── exercicios/
│   └── resolvidos/           # Exercícios e desafios propostos, já resolvidos
└── .gitignore
```

## 📌 Status

Repositório recém-criado, ainda nas primeiras aulas do curso. O conteúdo vai crescer conforme eu avanço nos módulos (variáveis, condicionais, laços, vetores/matrizes, sub-rotinas etc).

## 🎯 Objetivo

Fixar os conceitos fundamentais de lógica de programação — variáveis, tipos de dados, estruturas condicionais, laços de repetição, vetores/matrizes e modularização — antes de avançar para linguagens como JavaScript e Python.

## 🛠️ Como executar

Os algoritmos foram escritos e testados no app mobile **Pseudocode**, usando o dialeto **VisuAlg:Português**, direto pelo celular.
